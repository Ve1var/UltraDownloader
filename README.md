
### Загрузка видео
1. Выберите опцию `1`
2. Вставьте URL видео при появлении запроса
3. Следите за индикатором выполнения и дождитесь завершения

### Загрузка музыки
1. Выберите опцию `2`
2. Выберите режим загрузки:
   - `1`: Только один трек
   - `2`: Весь плейлист/альбом
3. Вставьте URL музыки
4. Аудио будет автоматически преобразовано в MP3

### Проверка обновлений
1. Выберите опцию `u` в главном меню
2. AutoUpdater проверит наличие новых версий
3. Следуйте инструкциям для загрузки и установки обновлений

## Организация файлов

Все загрузки сохраняются в структурированном формате:
```FileOrganization
Ваш_выбранный_каталог/
└── UD_Downloaded/
├── Video/
│ ├── Название видео YouTube.mp4
│ └── Название видео VK.mp4
└── Music/
├── Название песни.mp3
└── Название_плейлиста/
├── Трек 1.mp3
├── Трек 2.mp3
└── ...
```
## Важные замечания

### Юридические аспекты
- Загружайте только тот контент, на который у вас есть разрешение
- Соблюдайте законы об авторском праве и условия предоставления услуг
- Этот инструмент предназначен только для личного использования

### Ограничения
- Музыка ВКонтакте не поддерживается
- На некоторых платформах могут быть ограничения на загрузку
- Загрузка плейлистов зависит от поддержки исходной платформы

### Устранение неполадок

#### "ffmpeg.exe не найден"
- FFmpeg необязателен для базовых загрузок
- Укажите путь к FFmpeg через опцию `d` -> `1` в главном меню для конвертации аудио
- Или используйте встроенный FFmpeg из архива релиза

#### Ошибка загрузки
- Проверьте подключение к Интернету
- Убедитесь, что URL указан правильно и доступен
- Убедитесь, что у вас есть права на запись в каталог загрузки

#### Проблемы с обновлением
- Убедитесь, что `AutoUpdater.exe` находится в одной папке с `UltraDownloader.exe`
- Закройте основное приложение перед установкой обновлений

## Технические детали

- **Создано с использованием**: Python, yt-dlp, Tkinter
- **Конфигурация**: JSON в `%APPDATA%\UltraDownloader\app_config.json`
- **Зависимости**: FFmpeg (опционально)
- **Платформа**: Windows (автономный EXE с автообновителем)
